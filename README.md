# 🎬 Drama Diary

> 你的私人影视日记本

Drama Diary 是一个精心设计的个人影视记录应用，帮你记录看过的每一部好剧，留下那一刻的真实感悟。

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ✨ 功能特点

### 📝 影视档案管理
- 记录电影、电视剧、综艺等各类影视作品
- 自动获取影视信息（海报、演员、简介等）
- 支持本地上传海报图片

### 🏷️ 智能分类
- **在看** - 正在追的剧集
- **已看完** - 完整观看记录  
- **想看** - 收藏想看的片单

### ⭐ 评分与感悟
- 5 级情绪评分（从 😞 到 😍）
- 记录个人感悟与心得
- 添加自定义标签

### 🔍 快速查找
- 搜索片名、演员
- 按评分排序
- 按年份分组查看

## 📱 适合谁用？

- 🎭 **追剧达人** - 记录追剧进度，管理片单
