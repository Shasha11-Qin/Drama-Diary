# 🎬 Drama Diary

> 你的私人 AI 影视日记本

Drama Diary 是一个精心设计的个人影视记录应用，融合 AI 能力，帮你记录看过的每一部好剧，留下那一刻的真实感悟，并将抽象的情感转化为具象的视觉卡片。

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ✨ 核心功能

### 📥 多源录入
- **TMDB 搜索录入** - 一键搜索全球影视数据库，自动填充海报、演员、简介等信息
- **搜索结果筛选** - 按"电影"或"电视剧"分类筛选搜索结果
- **手工录入** - 对于 TMDB 找不到的内容，支持完全手动填写

### ✍️ AI 情感记录助手
- **AI 文本润色** - 将口语化的草稿润色成优美、有文学性的文字
- **AI 标签提取** - 自动分析感悟情感，推荐 2-4 个情感标签（如治愈、意难平）
- **AI 视觉情绪卡片** - 基于感悟情绪，从 TMDB 剧照库筛选契合氛围的背景，一键生成分享海报

### 📝 沉浸式写作体验
- **类 Notion 富文本编辑** - 支持粗体、斜体、标题、引用、列表
- **全屏写作模式** - 点击感悟区域弹出大尺寸编辑弹窗，沉浸式记录

### 📊 分集进度追踪
- **进度条可视化** - 卡片上直观显示观看进度
- **快速更新** - 滑块调整或 "+1集" 按钮一键更新
- **状态管理** - 想看 / 在看 / 看过 三种状态切换

### 🏷️ 结构化私人字段
- 刷过次数、专属海报、阶段性喜爱程度
- 自定义情感标签
- 首次观看时间、播出平台等

### 📊 智能展示
- **状态筛选** - 按想看/在看/看过切换
- **排序方式** - 看过页面支持按喜爱程度或年份分组

### 📥 数据迁移
- **豆瓣数据导入** - 支持豆瓣导出的 Excel 文件一键导入

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 6 |
| 后端服务 | Supabase (数据库 + 认证 + Edge Functions) |
| CSS 框架 | Tailwind CSS 4 |
| 动画 | Motion |
| 富文本编辑 | TipTap |
| AI 服务 | 智谱 GLM-4-Flash (免费) |
| 图片生成 | html-to-image |
| 影视数据 | TMDB API |
